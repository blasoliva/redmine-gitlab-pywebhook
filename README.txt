Overview
========

Repository of python webhook to integrate gitlab to Redmine issue tracker.

Configuration
=============

You must configure api_key and url in *config.ini* file:

Example
-------
: api_key: abcdefghijklmnopqrstuvwxyz0123456789012
: url: https://redmine.org
